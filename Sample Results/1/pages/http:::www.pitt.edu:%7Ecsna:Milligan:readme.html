The clustering software from Glenn Milligan ( gmilliga@magnus.acs.ohio-state.edu) comprises of three sets:
  1. Cluster Generation Software from Milligan (1985 Psychometrika):
    • CLUSTGEN.EXE - a DOS based execute file for the cluster generator
    • CLUSTGEN.FOR - Fortran source code for the cluster generator

  2. Hierarchical Clustering with Influence Detection:
    Cheng and Milligan (1996 Journal of Classification)
    Cheng and Milligan (1996 Multivariate Behavioral Research)
    Cheng and Milligan (1995 Education & Psychological Measurement)
    • HCINFLU.EXE - DOS based execute file for hierarchical clustering
    • HCINFLU.F - Fortran source code for clustering with influence detection
    • TEST1.DAT - Example data set
    • README.HC - Extensive documentation file for HCINFLU

  3. K-Means Clustering with Influence Detection:
    Cheng and Milligan (1996 Education & Psychological Measurement)
    • KMINFLU.EXE - DOS based execute file for k-means clustering
    • KMINFLU.F - Fortran source code for clustering with influence detection
    • TEST1.DAT - Example data set
    • CENT.DAT - Example centroid file
    • README.KM - Extensive documentation file for KMINFLU
Problems downloading. If you encounter problems downloading the files above, try saving the file to disk (In Netscape, hold down the mouse button over the link until the menu appears, then select "Save this link as..."). The files are can also be downloaded from ftp.pitt.edu/group/csna/Milligan.
