                    <h3>The Command Line Crash Course</h3>
                    <p>This is a mini-book that teaches you Unix <em>or</em> Windows command
                    line skills. Unlike the "Learn The Hard Way" series, this crash course is
                    designed to get you mostly capable in a few hours to a few days.  It has
                    little explanation and focuses more on doing a small set of commands until you
                    remember them.
                    </p>

                    this book will get you over that hump quickly.  You won't be a grand master, but
                    you'll be at least able to do the very basics.
