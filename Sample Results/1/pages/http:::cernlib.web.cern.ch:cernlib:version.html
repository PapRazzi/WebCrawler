Select your architecture and operating system. Once you select the package you should save it to your own file system. The compressed tar files can be extracted with the following commands:
  1. gunzip -c package.tar.gz | tar xf -
  2. On Windows 98/NT/2000 use Winzip
  3. Windows XP uncompresses automatically
Your browser may support uncompressing the files on-the-fly.
