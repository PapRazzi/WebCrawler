      <h2 class="grid_8 prefix_2 suffix_2">What Is MacRuby?</h2>
<p class="grid_8 prefix_2 suffix_2">MacRuby is an implementation of Ruby 1.9 directly on top of Mac OS X core technologies such as the Objective-C runtime and garbage collector, the <span class="caps">LLVM</span> compiler infrastructure and the Foundation and <span class="caps">ICU</span> frameworks. It is the goal of MacRuby to enable the creation of full-fledged Mac OS X applications which do not sacrifice performance in order to enjoy the benefits of using Ruby.</p>
<hr class="grid_12">
<div class="grid_12">
<h2 class="grid_4 alpha">Get It</h2>
<h2 class="grid_4">Build It</h2>
<h2 class="grid_4 omega">Use It</h2>
<p class="grid_4 alpha">You can install the <a href="https://github.com/downloads/MacRuby/MacRuby/MacRuby%200.12.zip">latest stable release</a> or, if you are feeling adventurous, you can install one of the <a href="http://macruby.jp/files/nightlies">nightly builds</a>. See the wiki for <a href="https://github.com/MacRuby/MacRuby/wiki/Setting-up-MacRuby">more detailed instructions</a>.</p>
<p class="grid_4">If you&#8217;d rather build your own MacRuby, simply clone the MacRuby repository from Github and follow the instructions from the <a href="https://github.com/MacRuby/MacRuby/blob/master/README.rdoc"><span class="caps">README</span></a>.</p>
<p class="grid_4 omega">After installing, MacRuby is available from the command-line as <code>macruby</code>. It also comes with project templates for use with Xcode. <a href="https://github.com/MacRuby/MacRuby/wiki/Creating-a-simple-application">Create your first application today!</a></p>
