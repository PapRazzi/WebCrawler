Packages

The Conceptive Python SDK tries to include all the binary packages needed for building and deploying business applications.  It has setuptools and pip installed to make it easy to install any package available through the Python Package Index.

The Python SDK contains the needed packages for :
  • Database : database drivers and toolkits
  • GUI : Qt, PyQt, Matplotlib, QtDesigner
  • Security : PyCrypto, OpenSSL
  • Science : NumPy, BioPython, Sympy, H5Py, VTK
  • Finance : cdecimal, 0MQ
  • Development : Spyder IDE, nose, Cython, virtualenv, faulthandler
And much more, the complete list is available here.
