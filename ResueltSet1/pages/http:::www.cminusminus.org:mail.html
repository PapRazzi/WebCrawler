The C-- Mailing List

Owing to persistent spam problems, we are transferring the C-- mailing list to Mailman. Though we hate Mailman, it does seem to block spam. Subscription and other info can be found at https://cminusminus.org/mailman/listinfo/cminusminus.

If you prefer, you can read the list through the newsgroup gmane.comp.lang.c--, which uses the GMANE mail/news gateway.
