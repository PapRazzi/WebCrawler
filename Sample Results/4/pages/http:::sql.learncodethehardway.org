                    <h3>Introducing Learn SQL The Hard Way</h3>
                    <p>This book will teach you the 80% of SQL you probably need to use it effectively,

                    around building web, desktop, or mobile applications because you don't know SQL, then
                    this book is for you.  It is written for people with no prior database, programming, or
                    SQL knowledge, but knowing at least one programming language will help.
